HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Paragraphs are indicated by P starting from 1-6 for example P1 P2 surrounded by <>
  • li or ol is to indicate bullet points
  • !DOCTYPE is not case-sensitive because it is a declaration. Declarations are not case-sensitive; HTML elements are case-sensitive
  • lang="en", which tells the browser the language for the webpage
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users. Metadata is typically not seen by users, but it contains important information about accessibility, search engines, and performance
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The head element in this case defines the following three elements: The charset element sets our character encoding. The http-equiv element sets an HTTP header for content. The viewport element sets the default view for our webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When using an external style sheet, we link the file using a link element placed in the HTML's head
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS
  • The first declaration addresses the width of the element at 80% of the available width
  • The box-shadow CSS property adds shadow effects around an element's frame
  • The border property assigns a border with a width of 5px that is solid and gray
  • The margin property assigns 40px to the top and bottom, and the auto value to the left and right

Git

  • git status: checks what branch we are currently on
  • git checkout -b \branch-name\: creates a new branch and switches to it
  • git checkout main: navigate to the main branch by entering the following Git command
  • git pull origin main: pull the remote main branch into the local main branch by entering the following command
  • cd: command directory, code . :pulls up VS code, cd .. :goes back one directory
  • git add -A : to stage our changes
  • git commit -m “your message here” : to mark the changes we made in this session
  • git push origin feature/add-js: to add the directory to the main file

JavaScript

  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number
  • Control flow is the order in which a computer executes code in a script
  • A variable is a named container that allows us to store data in our code
  • To work with individual data items inside an array, we need to have one more piece of information: the location of the data item in the array. This location is identified with a unique number called an index
  • A FOR loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over. Developers must use another statement, such as the conditional IF statement that we just learned, to interrupt the control flow by executing a block of code a specific number of times
  • The first statement determines the starting point for our loop. Since we want to start at the beginning of the array, we declare a variable x and give it a value of 0 (because the first index of an array is always 0)
  • The second statement is the condition. As long as the number in the variable x is less than the length of the array, the loop will keep running
